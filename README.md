# Sentiment Analysis using NLP and Machine Learning

## Project Overview

This project performs sentiment analysis on movie reviews using Natural Language Processing (NLP) and Machine Learning.

The model predicts whether a review is:
- Positive
- Negative

---

# Technologies Used

- Python
- Pandas
- Spacy
- Scikit-learn
- NLP
- Jupyter Notebook

---

# NLP Preprocessing

- Lowercasing
- Removing HTML tags
- Removing punctuation
- Removing stopwords
- Stemming

---

# Feature Extraction

TF-IDF Vectorization

---

# Models Used

- Logistic Regression
- Naive Bayes
- Support Vector Machine (SVM)
- Decision Tree
- Random Forest Classifier

---

# Best Accuracy

```text
0.85
```

IMDb Movie Reviews Dataset
